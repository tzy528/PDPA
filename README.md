# PDPA
PDPA is an academic project that aims to explain data poisoning methods based on diffusion models.
the files in the package models are class files, such as the DNN (MLP), diffusion.
if you want to generate fake users, run main.py for training, then run generate.py for generating.
In the generate.py file, parameter 1 represents the generation invocation distribution, and parameter 0 represents the generation of QoS.
The default hyper-parameter is set for WS-DREAM dataset (RT), and the dataset can be found at: https://wsdream.github.io/
